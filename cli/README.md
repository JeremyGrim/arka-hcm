HCM CLI (spécification HCM_013)
================================

Ce répertoire est réservé à une future implémentation de la CLI `hcm`.

Pour l'instant, la logique HCM est exposée via la lib TypeScript `HcmService`
(`src/hcmService`), conforme aux spécifications HCM_010, HCM_011, HCM_012 et HCM_014.
