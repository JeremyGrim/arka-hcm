# arka-hcm
Version officielle du **HCM – Hybrid Collective Memory**, la mémoire collective opérationnelle d’une équipe mixte Humains + Agents Arka. Le HCM est un système **déterministe**, **structuré**, **gouverné**, indépendant de l’OS, et destiné à capturer **la connaissance stable** et **l’état courant** nécessaires au travail d’équipe.
